## Backend of the Snake game

### Installation for development! 
For this project is setup with Node v24.11.1 and npm 11.6.2

1. `npm i`
2. `npm run dev`
3. Then everytime you change any of the files in `/src`, backend will automatically recompile and start the server again
4. Setup `BiomeJS` plugin for VsCode: [LINK](https://marketplace.visualstudio.com/items?itemName=biomejs.biome)
5. Whenever you do `CTRL+S` BiomeJS automatic formatting should now run (for TypeScript files in `/src` directory)
